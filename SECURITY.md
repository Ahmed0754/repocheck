# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in repocheck, please **do not** open a public GitHub issue.

Instead, email **syedahmedali.0520@gmail.com** with:
- A description of the vulnerability
- Steps to reproduce it
- Any potential impact

You can expect a response within 48 hours. Once the issue is confirmed and fixed, a public disclosure will be made.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
| 0.1.x   | No        |
